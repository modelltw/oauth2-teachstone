# Teachstone Provider for OAuth 2.0 Client

This package provides Teachstone OAuth 2.0 support for the PHP League's [OAuth 2.0 Client](https://github.com/thephpleague/oauth2-client).

## Installation

To install, use composer:

```
composer require teachstone/oauth2-teachstone
```

## Usage

Usage is the same as The League's OAuth client, using `\Teachstone\OAuth2\Client\Provider\Teachstone` as the provider.
